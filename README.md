# UniEvents 📱

A mobile application that helps students track and register for school club events in one place.

## ✨ Features

- 📍 **Browse Events** - Discover all upcoming club events in your school
- 📝 **Register for Events** - One-click registration for any event
- 🔔 **Event Notifications** - Get reminders before events start
- 🎯 **Club Management** - Clubs can create and manage their events
- 📅 **Event Tracking** - Keep track of your registered events

## 🛠️ Tech Stack

- **Frontend**: Dart + Flutter (95.8%)
- **Backend**: C++ (2.4%)
- **Build**: CMake (1.2%)
- **Other**: JavaScript, HTML, Swift

## 🚀 Getting Started

### Prerequisites
- Flutter SDK
- Dart
- Android/iOS development environment

### Installation

```bash
# Clone the repository
git clone https://github.com/ngoctuan653/uni_events.git
cd uni_events

# Install dependencies
flutter pub get

# Run the app
flutter run
```

## 📱 Usage

1. **For Students**:
   - Launch the app and browse available events
   - Register for events you're interested in
   - Enable notifications to get reminders

2. **For Clubs**:
   - Create your club account
   - Post new events with details and images
   - Manage registrations
